# DinoFiles 2: The Ultimate Cyber-Grade File Manager
DinoFiles 2 ialah aplikasi pengurus fail (File Manager) yang dibina khusus untuk pengguna yang mementingkan kecekapan, keselamatan, dan kawalan mutlak. Dibina dengan antaramuka berinspirasikan cyberpunk, aplikasi ini menggabungkan fungsi pengurusan fail asas dengan alatan keselamatan gred tinggi.

Ciri-Ciri Utama:
Navigasi Fail Pantas: Antaramuka ringkas untuk menyemak imbas fail di seluruh pemacu (Drive Switcher) dan akses pantas ke folder OneDrive.

Sistem Keselamatan AES-Encryption: Lindungi fail sensitif anda dengan fungsi Encrypt dan Decrypt menggunakan algoritma penyulitan Fernet yang teguh.

Padam Kekal (Secure Erase): Memintas Recycle Bin untuk memadam fail secara total dari cakera keras.

Kawalan Pemacu (Advanced Disk Tools): Keupayaan untuk memformat pemacu (Drive Formatting) secara terus dari aplikasi (memerlukan kebenaran pentadbir).

Log Aktiviti: Setiap tindakan yang dilakukan direkodkan dalam sistem log untuk tujuan audit dan pemantauan keselamatan.

Ringan & Efisien: Dibina menggunakan Python dengan kebergantungan (dependencies) minimum, menjadikannya sangat pantas dan stabil.

Siapa yang patut menggunakan DinoFiles 2?
DinoFiles 2 direka untuk pengguna kuasa (power users), pembangun perisian, dan individu yang mahukan alat pengurusan fail yang lebih daripada sekadar "Copy & Paste". Ia memberikan anda kawalan penuh ke atas data anda dengan pendekatan teknikal yang telus.

Amaran: DinoFiles 2 dilengkapi dengan alatan pengurusan cakera peringkat tinggi. Sila gunakan fungsi Format Drive dan Permanent Delete dengan penuh berhati-hati kerana data yang dipadam tidak boleh dipulihkan semula.

Maklumat Teknikal:
Bahasa: Python 3.x

Framework: Tkinter (GUI)

Keselamatan: Cryptography (Fernet)

DinoFiles 2: The Ultimate Cyber-Grade File Manager
DinoFiles 2 is a high-performance, cyberpunk-inspired file management utility designed for power users who prioritize efficiency, absolute control, and data security. Built for those who demand more than the standard OS file explorer, DinoFiles 2 integrates professional-grade security tools directly into your workflow.

Core Features
Intuitive File Navigation: Seamlessly switch between local drives and access your OneDrive cloud storage with a single click.

Military-Grade Encryption: Secure your sensitive files using the Fernet (AES-based) symmetric encryption standard. Encrypt and decrypt files instantly, keeping your private data out of unauthorized hands.

Permanent Data Destruction: Forget the Recycle Bin. Use the Secure Erase feature to permanently purge files and folders from your storage.

Advanced Disk Tools: Take full control of your storage with built-in drive formatting capabilities, designed for rapid hardware management.

Actionable Audit Logs: Every operation is meticulously recorded, providing a transparent history of your file management activity.

Ultra-Lightweight Architecture: Built with Python and minimal dependencies, ensuring a bloat-free, lightning-fast experience on any Windows machine.

Who is this for?
DinoFiles 2 is crafted for developers, system administrators, and security-conscious individuals who need a reliable, transparent, and powerful toolkit to manage their digital assets without the interference of unnecessary system processes.

Warning: DinoFiles 2 grants advanced system-level permissions. Functions such as Format Drive and Permanent Delete are irreversible. Always exercise caution when handling sensitive system data.
